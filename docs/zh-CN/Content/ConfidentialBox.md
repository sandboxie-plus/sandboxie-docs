# 机密沙盒

_ConfidentialBox_ 是 [Sandboxie Ini](SandboxieIni.md) 中的一个沙盒设置。

```
   .
   .
   .
   [DefaultBox]
   ConfidentialBox=y
```

使用 'ConfidentialBox=y' 选项可以防止主机进程对隔离进程进行读取访问。

**技术细节**

更多信息，请参见 [沙盒加密](../PlusContent/BoxEncryption.md) 和 [沙盒预设比较](../PlusContent/box-preset-comparison.md)。
