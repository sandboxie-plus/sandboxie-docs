    SBIE1409

    Message: SBIE1409 The Templates.ini file cannot be opened [xxxxxxxx]

    Logged To: Popup Message Log.

    Explanation:

    Sandboxie expects to find the global configuration file Templates.ini in its installation folder. This file should be considered a stock part of the installation and should not be edited or removed.

    See also: Nt Status Codes.
